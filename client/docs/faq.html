Frequently Asked Questions
I am getting compilation errors with Mirth; what do I do?
First, make sure that you are using an up to date version of Java. Mirth requires at least Java 5.0 to compile and run. Second, make sure that you have included all of the libraries that are required to compile Mirth. Otherwise, if you are certain that this is a bug in the source code, please forward the concern to mirth@mirthcorp.com.

How do I setup Java?
The way that you setup Java depends on what operating system you are currently using.

Please refer to this page to view installation instructions based upon your operating system.

I don't really understand what is needed to run Mirth, could you please clarify?
There are two parts to Mirth: a graphical administrative application and the underlying server application. The administrative application connects to the server application which contains all of the data and deals with actually processing the users commands. One server can potentially be accessed by many users at the same time. Both of these parts of Mirth are required to use its functionality.
